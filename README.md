##GRUPO 7 CURSO PROGRAMACION WEB FULL STACK 
## Nombre Sitio:
Arcolbooks

###DESCRIPCIÓN DEL SITIO:
El sitio se basa en un comercio electrónico para la venta de libros, el cual cuenta con un catálogo amplio de opciones y categorías para lectores principalmente niños, Jóvenes y adultos jóvenes.

###PÚBLICO OBJETIVO:
Los libros ofrecidos son para todo tipo de público, sin embargo, consideramos que la mayor demanda será de Niños, jóvenes y adultos jóvenes con edades entre los 6 y 35 años.
